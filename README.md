# TradesAdvisor Group FZ-LLC — Corporate Website

Строгий одностраничный сайт-визитка (Deep Tech, тёмная тема) для прохождения
банковского комплаенса в ОАЭ. Позиционирование — **независимый разработчик ПО и
провайдер образовательных технологий**; никаких финансовых услуг.

Главная страница `index.html` рендерится клиентским рантаймом `support.js`,
который сам подгружает React / ReactDOM / Babel с CDN — сборка не требуется.
Юридические страницы — обычная статика.

## Структура

| Файл | Назначение |
|------|-----------|
| `index.html` | Главная: Hero → About → Core Services → Technology & Portfolio → Contact + дисклеймер |
| `support.js` | Рантайм рендеринга (сгенерирован, не редактировать вручную) |
| `privacy.html` | Privacy Policy (полный текст) |
| `terms.html` | Terms of Use (полный текст) |
| `disclaimer.html` | Disclaimer (полный текст) |
| `legal.css` | Общие стили юридических страниц |
| `vercel.json` | Конфигурация статического деплоя |

Блоки главной точно соответствуют брифу «Сайт Визитка»: тёмная тема с медленно
движущейся абстрактной сеткой данных и полупрозрачным кодом на фоне, никаких
монет, свечей, стрелок и агрессивных призывов. Кнопки — «Request Consultation» /
«View Services». Форма контактов (Name, Email, Subject, Message) открывает
почтовый клиент через `mailto:`.

### Языки

Переключатель **EN / RU / AR** в шапке. Арабский — с полноценным RTL
(`dir="rtl"` + шрифт IBM Plex Sans Arabic). Весь текст интерфейса вынесен в
словарь переводов внутри `index.html` (метод `_dict()`), правится там же.

> Юридические страницы (`privacy.html`, `terms.html`, `disclaimer.html`) —
> **только на английском**: английский стандартен для комплаенса во фризонах ОАЭ,
> а переводы юридических текстов на RU/AR должен утвердить юрист. Пришлёте
> утверждённые переводы — добавлю переключатель и на них.

## Локальный запуск

Нужен HTTP-сервер (относительный `./support.js` через `file://` не работает):

```bash
npx serve .
# или
python3 -m http.server 3000
```

## Деплой на Vercel

Без фреймворка и build-шага — Vercel отдаёт файлы как есть.

```bash
npm i -g vercel
vercel --prod
```

Или импортом репозитория на vercel.com: Framework Preset — `Other`, Build Command
и Output Directory оставить пустыми.
