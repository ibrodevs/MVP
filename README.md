# Benchmark Technology Engineering L.L.C-FZ — Corporate Website

Статический корпоративный сайт компании по разработке ПО и инженерии данных.
Сайт позиционирует компанию через заказную разработку приложений, решения для
баз данных, интеграцию API, автоматизацию и кастомизацию программного
обеспечения.

## Структура

| Файл | Назначение |
|---|---|
| `index.html` | Главная: About, Services, Technology & Capabilities, Contact |
| `support.js` | Клиентский runtime главной страницы |
| `privacy.html` | Privacy Policy |
| `terms.html` | Terms of Use |
| `disclaimer.html` | Website Disclaimer |
| `legal.css` | Общие стили юридических страниц |
| `vercel.json` | Конфигурация статического деплоя |

Главная поддерживает английский, русский и арабский языки, включая RTL для
арабской версии. Юридические страницы опубликованы на английском в формулировках,
подготовленных для текущей корпоративной версии сайта.

Контактная форма собирает только Name, Email, Subject и Message, после чего
открывает почтовый клиент через `mailto:`. Текущий адрес получателя:
`projects@bteglobal.ae`.

Социальные ссылки и платёжные методы скрыты до появления подтверждённых
корпоративных профилей и реквизитов.

## Локальный запуск

```bash
npx serve .
# или
python3 -m http.server 3000
```

## Деплой на Vercel

Проект не требует сборки. При импорте репозитория выберите Framework Preset
`Other`, а Build Command и Output Directory оставьте пустыми.
